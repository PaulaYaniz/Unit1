## Last digit of integer
Given an integer number, print its last digit.
```.py
a = (input())

print(a[-1])
```

## Tens digit
Given an integer. Print its tens digit.
```.py
a = int(input())

print(a // 10 % 10)
```

## Sum of digits
Given a three-digit number. Find the sum of its digits.
```.py
a = int(input())

a1 = a // 100
a2 = a // 10 % 10
a3 = a % 10

print(a1+a2+a3)
```
