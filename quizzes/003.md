#### Create a program that translate the proteins in the DNA chain as shown below.

```.py
in_protein = input("Enter the protein: ")

if in_protein == "A":
    out_protein = "T"
elif in_protein == "G":
    out_protein = "C"
elif in_protein == "T":
    out_protein = "A"
elif in_protein == "C":
    out_protein = "G"

else:
    print("Error")

print(out_protein)
```
